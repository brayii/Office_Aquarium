Office Aquarium Failure Tester

This page creates a test save for the two failure paths in the main game. After choosing a test, open Office_Aquarium.html and press Continue Company. The simulation should trigger the failure screen within a few seconds of running.

These buttons write to the same browser save used by the game. Use Back Up Current Save before testing if you want to restore your current company afterward.

Test 1: CEO Removed by Board

Creates a save with board confidence at zero. The main game should show the Board Decision screen and say the board appointed a new CEO.

Test 2: Company Failure

Creates a save with cash below the insolvency threshold. The main game should show the failure screen and say the company ceased operations.

Regression Test Saves

These create targeted saves for systems that have broken or become confusing during long simulations. After creating one, open the main game and continue the company.

Open Main Game

Day Rollover Guard

Starts the game at 7:55 PM on day 70. The next tick should roll to the next day or pause with a recorded simulation error, not freeze.

Project Archive Integrity

Creates an old archived project and a live project with the same internal ID. The main game should preserve both by repairing the live project ID.

Near-Complete Project Closeout

Creates a 99.6% project with no blockers. The project should close cleanly instead of getting stuck forever below 100%.

Hiring Pipeline

Creates an active recruiting item and an onboarding employee. Workforce should show recruiting/onboarding instead of losing the new hire.

CEO Inbox Decision

Creates a queued memo with exactly three choices. The Inbox should open, pause the game, and let you choose one.

Commercial Revenue

Creates a completed launched archive project. Daily revenue should be greater than zero after the next daily close.

Local Unit Checks

These checks run on the test page itself. They do not simulate the whole game, but they catch bad saves and missing backup data before you open the main game.
